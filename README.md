# byAlif Apps

Public app registry for all apps by byAlif.

## Usage

The `apps.json` file is fetched by iOS apps to display a "More Apps by byAlif" section in Settings.

```
https://raw.githubusercontent.com/saroar/byalif-apps/main/apps.json
```
